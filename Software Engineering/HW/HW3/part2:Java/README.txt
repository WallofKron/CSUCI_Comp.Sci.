This java factory pattern designed is adopted from one of my old CS111 (College of the Canyons - community college) project, from when we were learning OOP. I changed a few things to show the OOP. But this design mirrors the Factory Pattern design.


MVC seems like a very valuable tool for building any sort of GUI application, as it lays out very clearly which classes have what responsibilities. And based on the structure, each class relies on the other two classes in a clear 2-way relationship.
